# Reference Packs

Drop local reference images here when you want image generation work to condition on existing art direction.

Recommended structure:

- `assets/art_direction/reference_packs/style/`
- `assets/art_direction/reference_packs/characters/<character_id>/`
- `assets/art_direction/reference_packs/stages/<stage_id>/`

Examples:

- `assets/art_direction/reference_packs/style/show_lighting_board.png`
- `assets/art_direction/reference_packs/characters/mark_zuck/octagon_pose_anchor.png`
- `assets/art_direction/reference_packs/stages/grand_arena/floor_material_board.png`

Keep source filenames stable so the same boards can be reused across imagegen iterations.
