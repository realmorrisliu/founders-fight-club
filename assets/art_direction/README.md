# Art Direction Inputs

This directory holds reusable creative inputs for direct image generation work.

- `style_lock.yaml`: shared visual rules and global constraints
- `character_briefs/`: per-character identity, silhouette, palette, prop, and rivalry inputs
- `stage_briefs/`: per-stage identity inputs
- `reference_packs/`: local mood boards, pose anchors, and material references

These files are intentionally separate from the runtime asset pipeline in `assets/sprites/` and the runtime integration guide in `docs/art/ART_PIPELINE.md`.
