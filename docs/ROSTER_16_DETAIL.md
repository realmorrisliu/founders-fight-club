# Launch 16 Detailed Character Sheets

This file expands the launch roster into implementation-ready content for writing, balancing, and presentation.

## 1. How To Use This File

- `Fantasy`: what the player should feel.
- `Visual Hook`: silhouette and prop direction for pixel art.
- `Passive`: identity rule that can be implemented with lightweight modifiers.
- `Signature A/B/C`: product-themed specials mapped to your current combat model.
- `Ultimate`: hype move for future meter system.
- `Rival Intro`: short pre-fight line idea.
- `Win Line`: post-fight one-liner.
- `Story Node`: where this character matters in arcade flow.
- `Clip Moment`: social-media-friendly highlight shot.
